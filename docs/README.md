# 사용 퀀트 / 금융 도메인

일봉 OHLCV (Open, High, Low, Close, Volume) — 데이터의 기본 단위
이동평균 (Moving Average) — 단순이동평균(SMA) vs 지수이동평균(EMA)
골든크로스 / 데드크로스 — 매수·매도 시그널 정의
백테스트 (Backtest) — 과거 데이터로 전략 성과 검증
수익률 계산 — 단순수익률 vs 로그수익률
MDD (Maximum Drawdown) — 최대낙폭, 리스크 지표
샤프지수 (Sharpe Ratio) — 수익률 대비 변동성, 전략 품질 지표
승률 (Win Rate) — 수익 거래 / 전체 거래
슬리피지 / 거래비용 — 백테스트 현실화 요소
