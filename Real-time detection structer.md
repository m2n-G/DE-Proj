MA 계산 /    :     Lambda (lambda_signal.py)  <---( 1분 주기 EventBridge 트리거 )
크로스 감지   |
               |   [ 작업 A: 리샘플링     ] --- 체결 데이터 → 1분봉 OHLCV 변환
               |                           --- 직전 20분치 1분봉 로드
               |
               |   [ 작업 B: MA 계산      ] --- pandas rolling(5).mean()  → MA5  (단기)
               |                           --- pandas rolling(20).mean() → MA20 (장기)
               |
               |   [ 작업 C: 크로스 감지  ] --- 전분 MA5 < MA20  &  현재 MA5 > MA20  →  golden_cross
               |                           --- 전분 MA5 > MA20  &  현재 MA5 < MA20  →  dead_cross
               |
               |   [ 작업 D: 분기 처리    ] --- 크로스 발생  →  SNS 발행 (즉시 알림 경로)
               |                           --- 크로스 없음  →  결과 저장 후 종료
               |
               +----( 크로스 발생 시 )-------( SNS Publish )----------------------> SNS → Slack 즉시 알림
               |
               +----( 계산 결과 저장 )-------( S3 PutObject )------------------------------------+
                                                                                                  |

분석 조회    :     Athena  --------( S3 Gold 파티션 SQL 조회 )
(사후 검증)    |
               |   -- 당일 골든크로스 발생 종목 조회
               |   SELECT stock_code, signal, close, MA5, MA20, datetime
               |   FROM signal_history
               |   WHERE date = '2024-01-15'
               |     AND signal = 'golden_cross'
               |   ORDER BY datetime;
               |
               |   -- 신호 발생 후 수익 여부 검증 (사후 분석)
               |   SELECT stock_code,
               |          signal_price,
               |          close_price,
               |          ROUND((close_price - signal_price) / signal_price * 100, 2) AS return_pct
               |   FROM signal_vs_close
               |   WHERE date = '2024-01-15';
               |
